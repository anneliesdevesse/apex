To use the B&K SLM 2250 plugin, you need to install
the B&K drivers first. Please proceed as follows:

* Download the software development kit
http://www.bksv.com/Home/Support/Downloads/2250%20Maintenance/Software%20Development%20Kit.aspx
* Download the utility software
http://www.bksv.com/Support/Downloads/2250%20Maintenance/Utility%20Software%20for%20Hand-held%20Analyzers.aspx

* Install both
* Run c:\program files\Bruel and Kjaer\\ENV\\BZ5503\\BasicEnvRemoteApi_Register.bat
